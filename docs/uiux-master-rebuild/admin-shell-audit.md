2. **Header consistency** ‚Äî admin-chrome PageHeader adoption is

3. **"No heading" is mostly a scan limitation** ‚Äî the 4
   admin-chrome routes with no detected heading are thin server wrappers whose

   `components/admin/publishing/PublishingOSClient`, `video-studio`). Verify
   before treating as a gap ‚Äî this scan only reads each route's own directory.
